# Admin Dashboard

Modern, responsive ve CRUD işlemleri destekleyen bir Admin Dashboard uygulaması.

Bu proje kullanıcı, müşteri, ürün ve sipariş yönetimi yapabilen bir yönetim panelidir. Uygulama React ile geliştirilmiş, veri yönetimi için Supabase kullanılmıştır.

## Features

- Staff management
- Customer management
- Product management
- Order management
- Create, read, update and delete operations
- Product stock control
- Order status tracking
- Payment status tracking
- Dashboard statistics
- Bar chart and pie chart visualizations
- Search and filter functionality
- User presence status
- Responsive design
- Loading, error and empty state handling
- Supabase database integration
